# Contributing to AgentPM

Thank you for your interest in contributing to AgentPM! We welcome contributions from the community to help make AgentPM the best package manager for autonomous AI agents.

## How to Contribute

1.  **Report Bugs:** If you find a bug, please open an issue with details on how to reproduce it.
2.  **Suggest Features:** Have an idea for a new feature? Open an issue and describe how it would benefit the ecosystem.
3.  **Submit Pull Requests:** Want to fix a bug or add a feature?
    *   Fork the repository.
    *   Create a new branch for your feature or bug fix.
    *   Make your changes and ensure tests pass.
    *   Submit a Pull Request with a clear description of your changes.

## Development Setup

1.  Clone your fork: `git clone https://github.com/your-username/agentpm.git`
2.  Install dependencies: `npm install`
3.  Build the project: `npm run build`
4.  Run tests (if applicable): `npm test`

## Code of Conduct

Please be respectful and considerate of others when participating in this project. We aim for a welcoming and inclusive environment.
